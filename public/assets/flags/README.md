# 国旗图片上传说明

## 目录结构
```
public/assets/flags/
├── README.md
├── en.png    # 英语
├── zh.png    # 简体中文
├── ja.png    # 日语
├── ko.png    # 韩语
├── de.png    # 德语
├── fr.png    # 法语
├── es.png    # 西班牙语
├── pt.png    # 葡萄牙语
├── ru.png    # 俄语
└── ar.png    # 阿拉伯语
```

## 图片规范

### 文件命名
- 文件名必须使用语言代码（ISO 639-1）
- 格式：`{language_code}.png` 或 `{language_code}.svg`
- 例如：`en.png`、`zh.png`、`ja.png`

### 图片尺寸
- 推荐尺寸：32x24 或 24x16（宽 x 高）
- 支持 PNG 和 SVG 格式
- 保持图片清晰度，避免模糊

### 背景要求
- 使用透明背景（PNG）或无背景（SVG）
- 不要使用带边框的方形图片

### 图片质量
- 优化文件大小，建议每个图片不超过 10KB
- 使用高质量的正版国旗图片

## 上传步骤

1. 准备10种语言的国旗图片：
   - en.png（英语 - 美国/英国）
   - zh.png（简体中文 - 中国）
   - ja.png（日语 - 日本）
   - ko.png（韩语 - 韩国）
   - de.png（德语 - 德国）
   - fr.png（法语 - 法国）
   - es.png（西班牙语 - 西班牙）
   - pt.png（葡萄牙语 - 葡萄牙/巴西）
   - ru.png（俄语 - 俄罗斯）
   - ar.png（阿拉伯语 - 阿拉伯国家）

2. 将图片上传到 `public/assets/flags/` 目录

3. 确保文件名与语言代码匹配

## 测试
上传完成后，刷新网站，语言选择器会自动显示对应国旗。

## 注意事项
- 请使用官方或高质量的国旗图片
- 确保国旗图片符合当地法律规定
- 避免使用有争议的国旗变体
