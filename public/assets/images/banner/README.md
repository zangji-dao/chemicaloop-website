# Banner 图片上传说明

## 目录位置
`public/assets/images/banner/`

## 上传步骤

### 1. 准备图片
请准备你的 banner 图片，建议：
- **格式**：JPG、PNG 或 WebP
- **尺寸**：建议 1920x800 或以上（宽度至少 1920px）
- **文件大小**：建议每张图片不超过 500KB（优化加载速度）
- **宽高比**：建议 16:9 或 21:9

### 2. 命名规范
按照以下格式命名图片：
- 如果只有 1 张：`banner-1.jpg`
- 如果有多张轮播：`banner-1.jpg`, `banner-2.jpg`, `banner-3.jpg` ...

### 3. 上传图片
将图片上传到当前目录：`public/assets/images/banner/`

### 4. 完成后告诉我
上传完成后，请告诉我：
- 上传了几张图片？
- 是否需要轮播效果？

我会立即修改代码并测试显示效果。

---

## 示例
```
public/assets/images/banner/
├── README.md
├── banner-1.jpg    ← 上传你的图片
└── banner-2.jpg    ← 如果有多张轮播
```

## 注意事项
- 图片内容应该清晰、高质量
- 建议使用压缩工具优化图片大小
- 避免使用过大文件影响加载速度
