# 代理大厅功能规划文档

## 一、概述

代理大厅是 Chemicaloop 平台为认证代理商提供的专属业务管理平台，核心功能包括产品管理和供应商报价查询。

---

## 二、功能模块

### 2.1 产品管理

代理商管理自己销售的产品，进行上下架操作。

#### 2.1.1 产品列表
- 展示已上架的所有产品
- 支持搜索：CAS码、产品名称
- 显示信息：
  - 产品名称、CAS码
  - 纯度、包装规格
  - 参考价格
  - 状态（上架/下架）
  - 操作（编辑、上下架）

#### 2.1.2 产品上架
| 字段 | 说明 | 必填 |
|------|------|------|
| CAS码 | 产品唯一标识 | ✓ |
| 产品名称 | 默认根据CAS自动填充，可修改 | ✓ |
| 纯度 | 如：99%、99.5%、99.9% | ✓ |
| 包装规格 | 如：25kg/桶、200kg/桶 | ✓ |
| 参考价格 | 元/kg | ✓ |
| 最小起订量 | 数量 | |
| 库存数量 | 可选是否公开 | |
| 产地 | 国家/地区 | |
| 备注 | 补充说明 | |

#### 2.1.3 产品操作
- 编辑产品信息
- 上架/下架切换
- 删除产品

---

### 2.2 供应商报价查询

根据CAS码查询该产品所有供应商的报价，方便寻找货源或比价。

#### 2.2.1 搜索功能
- 输入CAS码查询
- 支持模糊搜索
- 自动补全提示

#### 2.2.2 查询结果展示
显示该CAS码所有在售产品及其代理商信息：

| 字段 | 说明 |
|------|------|
| 代理商名称 | 可点击查看详情 |
| 所在地区 | 代理商地址 |
| 纯度 | 产品纯度 |
| 包装规格 | 包装方式 |
| 价格 | 参考单价 |
| 起订量 | 最小订购量 |
| 库存 | 现货/期货 |
| 联系方式 | 即时通讯方式 |

#### 2.2.3 操作
- 点击代理商名称查看详情
- 一键复制联系方式
- 发起询价（跳转消息系统）

## 三、页面结构

```
代理大厅
├── 产品管理
│   ├── 产品列表
│   └── 上架产品（弹窗/页面）
│
└── 报价查询
    ├── 搜索框
    └── 查询结果列表
```

## 四、数据表设计

### agents（代理商信息）
已存在用户表中，通过 role='AGENT' 标识

### agent_products（代理产品）
| 字段 | 类型 | 说明 |
|------|------|------|
| id | uuid | 主键 |
| agent_id | uuid | 代理商ID |
| cas | varchar | CAS码 |
| name | varchar | 产品名称 |
| purity | varchar | 纯度 |
| package_spec | varchar | 包装规格 |
| price | decimal | 参考价格 |
| min_order | int | 最小起订量 |
| stock | int | 库存数量 |
| stock_public | boolean | 是否公开库存 |
| origin | varchar | 产地 |
| remark | text | 备注 |
| status | enum | 状态：active/inactive |
| created_at | timestamp | 创建时间 |
| updated_at | timestamp | 更新时间 |

## 五、接口设计

### 产品管理
```
GET    /api/agent/products        # 获取产品列表
POST   /api/agent/products        # 上架产品
PUT    /api/agent/products/:id    # 更新产品
DELETE /api/agent/products/:id    # 删除产品
PATCH  /api/agent/products/:id/status  # 上下架
```

### 报价查询
```
GET /api/products/:cas/suppliers  # 根据CAS码查询所有供应商报价
```
